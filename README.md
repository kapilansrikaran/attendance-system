# Attendance System

## Setup and Run

### Server
1. Navigate to the `server` folder:
    ```bash
    cd attendance-system/server
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the server:
    ```bash
    npm start
    ```

### Client
1. Navigate to the `client` folder:
    ```bash
    cd ../client
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the client:
    ```bash
    npm start
    ```
